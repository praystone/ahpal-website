一、系統層級最佳化

SteamOS 基於 Arch Linux，但預設有許多背景服務與動畫會消耗資源。以下為實操步驟：

✅ 步驟 1：關閉不必要的動畫與特效

  1. 進入 設定 → 系統設定 → 桌面模式（若在遊戲模式，先切換至桌面）。
  2. 開啟「系統設定」→「工作區行為」→「桌面特效」。
  3. 關閉「啟用動畫」、「模糊背景」、「視窗縮放動畫」。
  4. 套用後重新登入，可節省約 5~10% 的 CPU 資源。

✅ 步驟 2：更新驅動與系統核心

保持 SteamOS 在最新穩定版：
設定 → 系統 → 系統更新 → 切換至「穩定版」並檢查更新
若你使用第三方 Proton 版本（如 Proton-GE），建議每月手動更新一次。

✅ 步驟 3：關閉背景連線與同步

  • 關閉 Steam 雲端同步（僅在需要時開啟）。
  • 關閉「遊戲內重疊」中的「啟用 Steam 社群」。
  • 在桌面模式停用不必要的開機啟動項（如 Discord、Spotify）。

三、功耗與續航調校

Steam Deck 的 APU 功耗範圍為 3W~15W。透過以下設定可大幅延長續航：

🔋 快速功耗設定（按 ⋯ 按鈕 → 效能）

  • 功耗限制： 輕度遊戲 (如《空洞騎士》) 設定 5W；3A 遊戲設定 10W~12W。
  • GPU 時脈： 手動鎖定在 800~1200 MHz 可避免電壓波動。
  • 幀率限制： 設定 30fps 或 40fps，比 60fps 省電 40%。

四、遊戲內設定實操步驟（以《柏德之門 3》為例）

不同遊戲引擎的最佳化方式略有差異，但以下通用步驟適用於多數 3D 遊戲：

  1. 進入遊戲前： 在 Steam 遊戲屬性中加入啟動參數 -dx11 或 -vulkan（視遊戲支援性）。
  2. 解析度縮放： 設定為 75% ~ 85%，並開啟 FSR 或 NIS。
  3. 陰影與反射： 設為「低」或「關閉」，這兩項對效能影響最大。
  4. 紋理品質： 維持「中」或「高」，因 Deck 的 VRAM 僅 8GB 共享。
  5. 抗鋸齒： 選擇 TAA 或 FXAA，關閉 MSAA。
  6. 垂直同步： 關閉，改由 Steam Deck 的幀率限制器控制。

完成後，你應該會看到幀率穩定在 30~40fps，且機身溫度低於 75°C。

六、常見問題（FAQ）

Q1：Steam Deck 需要安裝 Windows 嗎？
不需要。SteamOS 已針對掌機最佳化，且 Proton 相容層可執行 90% 以上的 Windows 遊戲。若你堅持要玩某些反作弊遊戲（如《天命 2》），則需雙系統。
Q2：設定完後遊戲反而變卡怎麼辦？
請確認是否開啟了「半速率著色」或「FSR 極致效能」，這兩項在 2D 或輕度遊戲中可能造成反效果。建議先還原為預設，再逐步調整。
Q3：如何快速切換設定檔？
在遊戲模式中按下 ⋯ 按鈕 → 效能 → 使用「每遊戲設定檔」功能，可為不同遊戲儲存專屬的功耗、幀率與 GPU 設定。
Q4：40Hz 更新率真的比 60Hz 省電嗎？
是的。根據 Valve 官方數據，40Hz / 40fps 比 60Hz / 60fps 節省約 30% 功耗，且視覺流暢度接近 50fps，是掌機甜蜜點。
Q5：長時間遊玩建議充電嗎？
建議開啟「電池保護模式」（設定 → 系統 → 電池 → 限制充電至 80%），可延長電池壽命。邊充邊玩時，功耗會自動由電源供應器負擔，不會損壞電池。

總結：打造你的專屬 Steam Deck

透過本指南的系統層級最佳化、顯示設定、功耗調校以及遊戲內實操步驟，你的 Steam Deck 將在效能與續航之間取得完美平衡。記得定期更新驅動與 Proton，並利用「每遊戲設定檔」功能，讓每款遊戲都跑在最適合的狀態。
