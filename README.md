## 📖 專案簡介

雅寶社區 · 頂客論壇是一個 AI 驅動的自動化內容平台，結合 Google Gemini 與 DeepSeek 雙 API，自動產出高品質繁體中文文章，並提供 39 款 HTML5 互動遊戲。

---

## 🏗️ 技術架構

| 元件 | 技術 | 說明 |
|------|------|------|
| 總指揮 | PowerShell | 流程控制與自動化部署 |
| 內容引擎 | Python 3 | AI 文章生成與品質檢查 |
| AI 模型 | Gemini + DeepSeek | 尖峰/離峰自動切換 |
| 託管 | Cloudflare Pages | CDN + 免費 SSL |
| 版本控制 | Git + GitHub | 原始碼管理 |
