【Mac 新手教學】剛從 Windows 轉過來？超實用 Mac 新手入門 12 個必學設定，效率直接翻倍！

發表時間：2024-11-20 | 作者：雅寶論壇資深編輯 (SEO 專欄) | 閱讀時間：約 8 分鐘
剛從 Windows 陣營跳槽到 macOS 系統，看著精緻的畫面卻不知道從何下手？為什麼按右鍵沒反應？為什麼拖移視窗這麼吃力？別慌！身為 20 年的資深 Mac 玩家，今天這篇文章將為大家整理出 Mac 新手入門 12 個必學設定。只要花 10 分鐘跟著步驟調整，保證讓你的 MacBook 變得跟手腳一樣靈活，生產力直接飆升！

為什麼 Mac 買來一定要先調整設定？

Apple 的 macOS 系統設計邏輯非常著重於「極簡」與「直覺」，但也因為如此，許多非常強大、好用的功能在預設情況下是被「關閉」或「隱藏」的。特別是對於習慣了 Windows 檔案總管與快捷鍵邏輯的新手來說，若不先進行優化，很容易覺得 Mac「很難用」。以下這 12 個設定，就是幫你打通 Windows 與 macOS 雙系統任督二脈的關鍵！

Mac 新手入門 12 個必學設定實操指南

1. 啟用觸控板「輔助按鈕」（就是右鍵！）

剛拿到 Mac 的人最崩潰的就是：為什麼按觸控板右側沒有右鍵選單？預設必須要用兩隻手指點按才行。其實你可以改成跟 Windows 一樣的右下角點按。

設定步驟：
點選左上角「蘋果 」選單 ➔「系統設定」 ➔ 「觸控板」 ➔ 在「點按」標籤頁中，找到「輔助按鈕」，將其下拉選單改為「按壓右下角」（或保留兩指點按，依個人習慣）。

2. 開啟「輕點來按一下」

預設的 Mac 觸控板需要實體「按壓」下去才有反應，用久了手指容易疲勞。開啟這個功能後，你只需要像滑手機一樣「輕輕碰觸」就能完成點擊。

設定步驟：
「系統設定」 ➔ 「觸控板」 ➔ 「點按」 ➔ 將「輕點來按一下」功能勾選開啟。

3. 解鎖隱藏神技：「三指拖移」

這是許多 Mac 老手一致認為最不可或缺的功能！預設要拖曳視窗或選取文字，必須用力按住觸控板再移動。開啟「三指拖移」後，你只要用三隻手指輕放在觸控板上，就能直接拖曳視窗、選取檔案或文字，極度省力！

設定步驟：
「系統設定」 ➔ 「輔助使用」 ➔ 「指標控制」 ➔ 點選「觸控板選項」 ➔ 開啟「使用觸控板進行拖移」，並將拖移風格下拉選單選擇「三指拖移」。

4. 設定「熱角 (Hot Corners)」快速啟動

熱角功能可以讓你把滑鼠游標移到螢幕的「四個角落」時，瞬間觸發特定動作（例如：鎖定螢幕、顯示桌面、啟動螢幕保護程式）。

設定步驟：
「系統設定」 ➔ 「桌面與 Dock」 ➔ 滑到最下方點選「熱角...」 ➔ 為螢幕四個角指定功能（推薦：右下角設為「桌面」，右上角設為「鎖定螢幕」）。

5. 顯示 Finder（檔案總管）的「路徑列」與「狀態列」

Mac 的 Finder 預設非常乾淨，但這會讓你找不到檔案具體存在哪個資料夾，也看不到硬碟還剩多少容量。開啟這兩個設定，資訊一目了然！

設定步驟：
打開任意 Finder 視窗 ➔ 點選頂部選單列的「顯示方式」 ➔ 點選「顯示路徑列」與「顯示狀態列」。

6. 修改預設瀏覽器與搜尋引擎

如果你習慣使用 Google Chrome，可以將其設為預設瀏覽器，這樣點擊郵件或訊息中的連結時，就會自動用 Chrome 開啟。

設定步驟：
「系統設定」 ➔ 「桌面與 Dock」 ➔ 找到「預設網頁瀏覽器」 ➔ 下拉選擇你下載好的 Chrome 或其他瀏覽器。

7. 調整「中英文輸入法」切換快捷鍵

Windows 切換中英文是按 Shift 或 Ctrl+Space。Mac 預設是按左下角的 caps lock ⇪（中/英鍵）。如果你不習慣，可以調整為傳統的 Control + Space。

設定步驟：
「系統設定」 ➔ 「鍵盤」 ➔ 點選「鍵盤快速鍵...」 ➔ 選擇「輸入來源」 ➔ 自訂你習慣的切換組合；或者在鍵盤設定中勾選/取消「使用中/英鍵切換...」。

8. 自動隱藏 Dock 工具列（釋放螢幕空間）

Mac 下方的 Dock 欄很佔空間，尤其是 13 吋的 MacBook。將它設定為自動隱藏，當滑鼠移過去時才顯示，能讓你的工作畫面變得寬敞許多。

設定步驟：
「系統設定」 ➔ 「桌面與 Dock」 ➔ 勾選「自動隱藏和顯示 Dock」。

9. 活用 Spotlight 萬能搜尋（最速啟動 App）

不要再慢慢去應用程式資料夾找軟體了！學會用 Command (⌘) + Space 喚醒 Spotlight，輸入軟體名稱（如 "Line"）按 Enter 就能秒開，它還能當計算機和匯率換算器使用。

操作方式：
隨時按下鍵盤上的 Command (⌘) + 空白鍵 (Space)，輸入你要找的檔案、App 或是計算式（例如：120*45）。

10. 開啟「最佳化電池充電」延長電池壽命

MacBook 的電池很貴！開啟此功能後，Mac 會學習你的每日充電習慣，暫緩充電超過 80%，有效減緩電池老化速度。

設定步驟：
「系統設定」 ➔ 「電池」 ➔ 點選「電池健康度」旁的 「ℹ️」 按鈕 ➔ 開啟「最佳化電池充電」。

11. 設定「專注模式」（勿擾模式）避免彈窗打擾

工作或開會時，最討厭右上角一直彈出 LINE 或郵件通知。設定專注模式可以一鍵安靜。

設定步驟：
點選螢幕右上角的「控制中心」圖示（兩個膠囊圖案） ➔ 點選「專注模式」 ➔ 選擇「勿擾模式」，或在「系統設定」➔「專注模式」中自訂允許通知的聯絡人與 App。

12. 啟用「接續互通相機」與「通用剪貼簿」

如果你同時擁有 iPhone，這絕對是神級功能。在 Mac 上複製文字，可以直接在 iPhone 上貼上；或者在 Mac 上寫文件時，直接調用 iPhone 相機拍照並即時插入文件中。

設定步驟：
確保 Mac 與 iPhone 登入同一個 Apple ID，並同時開啟 Wi-Fi 與藍牙。系統會自動啟用「通用剪貼簿」與「接續互通」功能，無需額外設定。

Windows 與 macOS 常用功能與快捷鍵對照表

為了讓剛從 Windows 轉過來的朋友能無痛過渡，小編特別整理了這張超實用的功能對應與快捷鍵轉換表：

功能/操作 Windows 系統 macOS 系統 新手注意事項
檔案總管 / 檔案管理 本機 / 檔案總管 Finder (訪達) Mac 沒有 C 槽 D 槽的概念，皆以資料夾管理。
複製 Ctrl + C Command (⌘) + C Mac 的 Command 鍵相當於 Windows 的 Ctrl。
貼上 Ctrl + V Command (⌘) + V 同上。
剪切檔案（移動） Ctrl + X Cmd + C ➔ Cmd + Option + V Mac 不能直接對檔案按 Cmd+X，必須在貼上時加上 Option。
開啟檔案/資料夾 Enter (確定鍵) Command (⌘) + O 或 Cmd + ↓ 在 Mac 按 Enter 是「重新命名」檔案！
強制結束程式 Ctrl + Alt + Del Command (⌘) + Option + Esc 當程式當掉、轉彩虹球時，用此組合鍵強制關閉。
刪除檔案 Delete Command (⌘) + Delete 單按 Delete 鍵在 Mac 上無反應。

💡 Mac 新手常見問題 FAQ

為什麼我的外接隨身碟/行動硬碟接上 Mac 只能看、不能寫入？
這是因為你的硬碟格式為 Windows 專用的 NTFS 格式，macOS 預設僅支援讀取、無法寫入。建議將硬碟格式化為 exFAT 格式（Windows 與 Mac 皆支援讀寫），或者在 Mac 上安裝第三方 NTFS 讀寫軟體（如 Paragon NTFS 或 Tuxera NTFS）。
Mac 需要安裝防毒軟體嗎？
一般情況下不需要。macOS 本身擁有非常嚴格的安全防護機制（如內建的 Gatekeeper 與 XProtect）。只要你不要亂點不明郵件連結、不下載盜版破解軟體（dmg 檔），光靠內建的安全機制就非常足夠了。
怎麼在 Mac 上進行螢幕截圖？
Mac 內建非常強大的截圖功能：
  • 全螢幕截圖： Shift + Command (⌘) + 3
  • 選取區域截圖： Shift + Command (⌘) + 4
  • 開啟截圖工具列（錄影/視窗截圖）： Shift + Command (⌘) + 5
Mac 關機需要每天做嗎？還是直接蓋上螢幕就好？
直接蓋上螢幕睡覺即可！ Mac 的電源管理極其優秀，休眠時耗電量極低（一整晚大約只耗 1-2% 系統電量）。頻繁開關機反而會消耗更多啟動電量與硬碟壽命。建議大約 1-2 週重啟一次系統，清除快取即可。
