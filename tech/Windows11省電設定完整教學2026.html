🔋 為什麼 2026 年 Windows 11 省電設定更重要？

2026 年的 Windows 11 已更新至 24H2 版本，雖然微軟持續改善電源效率，但新功能如 AI 助手（Copilot）、即時字幕、背景動畫等也增加了耗電。加上許多輕薄筆電採用 Arm 架構（如 Snapdragon X Elite），正確的省電設定能讓續航從 8 小時躍升至 12 小時以上。雅寶社區 · 頂客論壇 整理了目前最完整的調整方案，包含你從未注意的登錄檔最佳化。

⚙️ 基礎省電設定 (5 分鐘快速優化)

以下步驟適合所有 Windows 11 使用者，立即見效。

① 開啟「節能模式」

點擊工作列右下角電池圖示 → 滑塊拉至「最佳電源效率」。2026 年的 Windows 11 新增了「極致省電」模式（需在電源與電池設定中啟用）。

實操： 設定 → 系統 → 電源與電池 → 選擇「節能模式」或「最長續航」。

② 關閉背景應用程式

許多 UWP 應用（如新聞、天氣、Xbox）在背景持續耗電。前往：設定 → 應用程式 → 已安裝的應用程式 → 點選各應用 → 進階選項 → 背景應用程式權限設為「永不」。

③ 調整顯示器亮度與關閉時間

降低亮度至 40%~50%，並設定閒置 2 分鐘後關閉螢幕。路徑：設定 → 系統 → 電源 → 螢幕與睡眠。

④ 關閉動畫與透明效果

設定 → 協助工具 → 視覺效果 → 關閉「透明效果」與「動畫效果」。這對整合顯示晶片機種特別有效。

🧠 進階電源計畫與隱藏選項 (2026 最新)

基礎設定不夠？雅寶社區 · 頂客論壇 為你挖掘 Windows 11 深層電源選項。

🔧 啟用「終極效能」電源計畫 (適合桌機)

雖然「終極效能」主要用於高效能，但搭配限頻後反而更省電。以系統管理員執行 PowerShell：
powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61
然後在電源選項中選取，並編輯進階設定：處理器最大頻率設為 80%。

🔧 隱藏的「處理器效能提升模式」

電源選項 → 變更計畫設定 → 變更進階電源設定 → 處理器電源管理 → 處理器效能提升模式 → 設為「停用」或「有效率」。這能防止 CPU 瞬間升頻浪費電力。

🔧 登錄檔調整：關閉不必要的硬體排程

開啟登錄編輯程式，前往 HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Power，新增 DWORD EnergyEstimationEnabled 設為 0，可讓系統更積極省電。⚠️ 請先備份登錄檔。

💻 硬體與周邊調校 (筆電/桌機通用)

軟體設定之外，硬體調校能再省 15%~20% 電力。

  • 關閉未使用的連接埠：裝置管理員 → 停用閒置的藍牙、乙太網路、Wi-Fi 介面卡。
  • 調整處理器最大狀態：電源進階設定 → 處理器最大頻率設為 99% (筆電) 或 80% (桌機)。
  • 啟用 SSD 節能：在電源選項中將「硬碟」→「關閉硬碟」設為 5 分鐘。
  • 使用省電螢幕更新率：設定 → 系統 → 顯示器 → 進階顯示 → 選擇 60Hz 或更低 (如 48Hz)。

❓ 常見問題 FAQ (Windows 11 省電)

Q1：開啟省電模式後，電腦變很慢怎麼辦？

可個別調整處理器最大頻率，設為 80%~90% 保留流暢度。另建議關閉省電模式中的「降低螢幕更新率」選項。

Q2：桌機也需要省電設定嗎？

當然！尤其長時間開機的桌機，每年可節省數百度電費。建議啟用「終極效能」並限制 CPU 頻率，搭配 SSD 休眠。

Q3：2026 年的 Windows 11 有沒有內建省電建議？

有！設定 → 系統 → 電源與電池 → 點選「省電建議」，系統會自動列出可關閉的耗電功能 (如背景活動、動畫)。

Q4：Arm 架構的 Windows 11 省電設定不同嗎？

基本邏輯相同，但 Arm 機種建議開啟「Microsoft 省電排程器」並關閉 x64 模擬 (若無需要)。雅寶社區 有專文討論。

Q5：關閉硬體加速排程會影響遊戲嗎？

輕度遊戲影響不大，3A 大作建議保留。可在電源進階設定中為特定應用程式建立自訂電源計畫。
